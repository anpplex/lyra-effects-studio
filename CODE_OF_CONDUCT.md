# Contributor Covenant Code of Conduct

We pledge to make participation in this project a harassment-free experience for everyone. Be respectful, assume good intent, welcome constructive feedback, and focus discussion on the work.

Project maintainers may remove, edit, or reject contributions and communication that violate these principles. Report unacceptable behavior privately through the security or maintainer contact published in this repository.

This policy is adapted from the Contributor Covenant, version 2.1.
